== Description ==

Modern browsing shows a non-instrusive and cool bar at the top of the page depending on the browser. The difference, you set which browser is obsolete for your site.

It supports:

*   Safari
*   Firefox
*   Opera
*   Internet Explorer

Also has:

*   HTML5 Shiv for IE (administration option)
*   The hability to close the notice by user

== Installation ==


1. Upload `modern-browsing` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure it in Options/Modern-Browsing
